import { HT } from '@/components/Highlight/index.ts';

:::note
<HT>`[[stateName]]`</HT> notifies that its `value` has changed when the previous `value` is different from the current `value`.
If its `value` is an <HT>`Object`</HT>, it does not detect internal changes, only when `value` is setted to another <HT>`Object`</HT>.
Learn more about [updating objects in state](/reactter/learn_more/updating_objects_in_state).
:::
