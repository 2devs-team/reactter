## Instalación

Para seguir esta guía, necesitarás un proyecto Dart/Flutter existente.
Si aún no tienes uno, puedes crear una app Dart <a href="https://dart.dev/tutorials/server/get-started" target="_blank">aquí</a> o una app Flutter <a href="https://docs.flutter.dev/get-started/install" target="_blank">aquí</a>.

Una vez configurado el proyecto, proceda a instalar el paquete utilizando uno de los siguientes métodos:

- [Instalación automatizada](#instalación-automatizada)
- [Instalación manual](#instalación-manual)

### Instalación automatizada

Añada el paquete a su proyecto ejecutando el siguiente comando:

<Tabs groupName="project_type">
  <TabItem name="dart" label="Solo Dart" icon="dart">
    ```shell showLineNumbers=false
    dart pub add reactter
    ```
  </TabItem>
  <TabItem name="flutter" label="Con Flutter" icon="flutter">
    ```shell showLineNumbers=false
    flutter pub add flutter_reactter
    ```
  </TabItem>
</Tabs>

### Instalación manual

Añada el paquete a su proyecto añadiendo la siguiente línea a su archivo `pubspec.yaml`:

<Tabs groupName="project_type">
  <TabItem name="dart" label="Solo Dart" icon="dart">
    <Code
      title="pubspec.yaml"
      lang="yaml"
      showLineNumbers={false}
      code={`
dependencies:
  reactter: ${reactterVersion ? `^${reactterVersion}` : "# add version here"}
      `}
    />
  </TabItem>
  <TabItem name="flutter" label="Con Flutter" icon="flutter">
    <Code
      title="pubspec.yaml"
      lang="yaml"
      showLineNumbers={false}
      code={`
dependencies:
  flutter_reactter: ${flutterReactterVersion ? `^${flutterReactterVersion}` : "# add version here"}
      `}
    />
  </TabItem>
</Tabs>

Y ejecute el siguiente comando para instalar el paquete:

<Tabs groupName="project_type">
  <TabItem name="dart" label="Solo Dart" icon="dart">
    ```shell showLineNumbers=false
dart pub get
    ```
  </TabItem>
  <TabItem name="flutter" label="Con Flutter" icon="flutter">
    ```shell showLineNumbers=false
flutter pub get
    ```
  </TabItem>
</Tabs>

## Configuraciones adicionales

Este paso es opcional pero recomendable.
Utilice el paquete <span class="inline-block align-middle">[![Reactter Lint](https://img.shields.io/pub/v/reactter_lint?color=1d7fac&labelColor=29b6f6&label=reactter_lint&logo=dart)](https://pub.dev/packages/reactter_lint)</span> para aplicar las mejores prácticas y convenciones en su proyecto::

```shell showLineNumbers=false
dart pub add reactter_lint
```

Además, si estás desarrollando con Visual Studio Code, es una buena idea utilizar [Reactter Snippets](https://marketplace.visualstudio.com/items?itemName=CarLeonDev.reacttersnippets) para mejorar la productividad.

## Uso

Para utilizar Reactter, debe importar el paquete en su archivo Dart donde lo necesite:

<Tabs groupName="project_type">

    ```dart
import 'package:reactter/reactter.dart';
    ```
  </TabItem>
  <TabItem name="flutter" label="Con Flutter" icon="flutter">
    ```dart

    ```
  </TabItem>
</Tabs>

Listo! No tienes que preocuparte por la configuración, Reactter ya esta listo para usarse en su proyecto.
